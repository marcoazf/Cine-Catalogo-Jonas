Manual do Catálogo — CineCatalog Elo

Versão 25.9 — Guia completo de funcionalidades e dicas de uso

1. Visão Geral

O CineCatalog Elo é um sistema premium de gerenciamento de acervo pessoal de filmes, séries e estreias. Desenvolvido em JavaScript puro (SPA) e empacotado com Electron, ele funciona 100% offline e armazena todos os dados localmente no navegador.

v25.9 Electron JS SPA Offline First

Características principais:

  • Cadastro completo de filmes e séries com capa, sinopse, elenco, diretor, género, estrelas e status
  • Sistema de estreias com datas de lançamento e notificações automáticas
  • Pesquisa booleana com + para combinar termos
  • Múltiplos modos de visualização: carrossel, grelha, marquee e dinâmico
  • Backup e restauro em formato JSON
  • Auto-salvamento e sugestão diária de filmes/séries
  • Suporte a Smart TVs com navegação por controle remoto (DPAD)
  • Reprodução de mídia no player padrão do sistema operacional

3. Cadastro de Filmes e Séries

Clique em + NOVO para abrir o formulário de cadastro. A janela possui duas abas: Filmes e Séries, além da aba de Estreias para lançamentos futuros.

Modo de Edição

Clique com o botão direito em qualquer card ou use o botão Editar na janela INFO. Os dados são carregados automaticamente nos campos. O botão muda para ATUALIZAR.

Séries — Cadastro Dinâmico

Na aba Séries, use o botão CADASTRO DINÂMICO para criar múltiplas temporadas e episódios de uma só vez. Defina o número de temporadas e a quantidade de episódios por temporada.

4. Estreias

A aba Estreias permite acompanhar lançamentos futuros e passados. As estreias são ordenadas por data: as futuras aparecem primeiro (da mais próxima para a mais distante), seguidas das passadas (da mais recente para a mais antiga).

  • Estreias vencidas são removidas automaticamente após 3 dias
  • Notificações pop-up são exibidas no dia da estreia
  • Notificações podem ser ativadas/desativadas nas Configurações

5. Pesquisa e Filtros

Pesquisa Simples

Digite na barra de busca (Ctrl+F) para filtrar por título, diretor, elenco, sinopse ou género. O resultado é instantâneo e os cards correspondentes ganham um brilho neon laranja.

Pesquisa Booleana

Use o operador + para combinar múltiplos termos. Exemplo: ação+2024+willis encontra filmes de ação de 2024 com Bruce Willis.

Pesquisa por Voz

Clique no ícone de microfone na barra de busca para ativar a pesquisa por voz (navegadores compatíveis).

Filtros

O painel de filtros (ícone fa-filter) oferece:

  • Geral: Todos, Recente, Antigos, Ordenação A-Z
  • Ano: Filtrar por ano de lançamento
  • Status: Novo, Assistir, Favoritos
  • Gêneros: Filtrar por categoria específica
Dica: Use a pesquisa booleana para encontrar combinações específicas. Ex: "terror+2024" mostra apenas filmes de terror de 2024.

7. Gestão de Gêneros

Os gêneros (categorias) podem ser geridos de duas formas:

  • Botão de engrenagem (fa-cog) ao lado do campo Gêneros no formulário de cadastro
  • Configurações → Personalização dos Cards → Gerenciar Gêneros: Painel completo com input para adicionar e tags para remover gêneros em tempo real

As alterações refletem-se automaticamente em todos os dropdowns do aplicativo, ordenados alfabeticamente.

9. Dashboard (Estatísticas)

O painel de dashboard apresenta:

  • Gráfico de distribuição por género (barras horizontais)
  • Gráfico de filmes por ano (barras ou colunas)
  • Totais: filmes, séries, estreias
  • Itens favoritados, novos, para assistir
  • Episódios totais (séries)
  • Gráfico em donut da proporção Filmes vs Séries

10. Sugestão do Dia

O sistema pode sugerir um filme ou série aleatório todos os dias. Ative nas Configurações e escolha os filtros desejados:

  • Novo — Itens marcados como "Novo"
  • Assistir — Itens marcados como "Para Assistir"
  • Favoritos — Itens favoritados

A sugestão aparece automaticamente ao carregar o aplicativo (apenas uma vez por dia). Também pode ser acionada manualmente pelo botão "Nova Sugestão" no modal.

Ao clicar em ASSISTIR, o filme/série é reproduzido no player padrão do sistema operacional, igual aos cards.

11. Execução de Mídia

O CineCatalog Elo suporta reprodução de vídeos diretamente no player padrão do sistema:

  • Clique no ícone de Play sobre o card ou no botão EXECUTAR na janela INFO
  • No Electron, o arquivo é aberto com o player padrão do Windows (ou sistema configurado)
  • URLs de trailer são abertas no navegador padrão
  • Suporte aos formatos: MP4, AVI, MKV, MOV, WMV, FLV, WEBM
Dica: Configure o caminho da pasta de vídeos em Configurações → Caminhos para abrir o seletor de ficheiros já na pasta correta.

12. Lembretes

Adicione lembretes personalizados a qualquer filme ou série. Use o botão de notas no card ou na janela INFO. Os lembretes são exibidos no painel combinado (Filmes + Séries juntos) com numeração sequencial e ícone identificador do tipo.

13. Backup e Restauro

  • Exportar (Ctrl+E): Baixa um arquivo JSON com todos os filmes, séries, categorias e configurações de tema
  • Importar (Ctrl+I): Substitui todos os dados atuais pelo conteúdo do arquivo JSON selecionado
  • O caminho de backup pode ser configurado em Configurações → Caminhos → Backups
  • No Electron, o exportador tenta usar a pasta de Backups configurada automaticamente
Dica: Faça backups regulares do seu acervo. O auto-salvamento também pode ser configurado para exportar automaticamente para a pasta do acervo.

14. Auto-salvamento

O sistema pode salvar automaticamente todos os dados em intervalos regulares. Configure:

  • Ativar/Desativar em Configurações → Auto Salvamento
  • Pasta do Acervo: Se configurada, o auto-salvamento também exporta o JSON para essa pasta
  • O indicador no rodapé mostra o status do salvamento automático

15. Suporte a Smart TV

O CineCatalog Elo é totalmente funcional em Smart TVs:

  • Android TV: Navegação por controle remoto com setas (DPAD), suporte a touch
  • Panasonic Smart TV (WebOS / não Android): Interface adaptada para resoluções Full HD (1920x1080)
  • Cards com foco visível (outline laranja) para navegação por teclado/controle
  • Navegação entre cards usando as setas do teclado/controle remoto

17. Dicas de Uso

  • Organização: Use os status (Novo, Assistir, Favorito) para categorizar o que já viu e o que pretende ver
  • Avaliação: Atribua estrelas aos filmes que já assistiu para criar um histórico de preferências
  • Pesquisa booleana: Combine termos com + para pesquisas precisas (ex: ficção+científica+2024)
  • Cadastro rápido: Use o botão direito do rato sobre um card para opções rápidas de editar, favoritar, lembrete ou eliminar
  • Géneros personalizados: Crie os seus próprios gêneros nas Configurações — eles aparecem em todos os dropdowns
  • Multi-monitor: O Electron abre maximizado, adaptando-se a qualquer resolução
  • Atualizações: Clique no badge de versão no rodapé para verificar se há nova versão disponível no GitHub
